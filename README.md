# Age Of Empires Units-React-Redux-saga
This is an app to filter age of empires units by the costs and ages. Building with ReactJS.
## To run the project
```
npm install
```
```
npm start
```
```
npm run build
```
## Used Technologies
<li>React</li>
<li>React-Bootstrap</li>
<li>React Routers</li>
<li>Material UI</li>
<li>React Redux</li>
<li>Redux-Saga</li>
